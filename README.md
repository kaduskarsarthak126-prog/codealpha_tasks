## Project 1: codealpha_PasswordStrengthChecker

### Project Description
This Java application checks the strength of a password based on different
validation rules such as length, uppercase letters, lowercase letters, numbers,
and special characters.

### Features
- Checks minimum password length
- Validates uppercase and lowercase letters
- Checks numeric digits
- Checks special characters
- Displays password strength as:
  - WEAK
  - MEDIUM
  - STRONG

### How to Run
1. Open terminal in the project folder
2. Compile the program:
   javac PasswordStrengthChecker.java
3. Run the program:
   java PasswordStrengthChecker

### Output
The program asks the user to enter a password and displays its strength.

## Project 2: codealpha_QuizApp

### Project Description
This is a console-based quiz application developed using Java. The application
asks multiple-choice questions to the user and calculates the final score based

### Features
- Multiple-choice questions (MCQs)
- User input through console
- Automatic score calculation
- Simple and beginner-friendly logic

### How to Run
1. Open terminal in the project folder
2. Compile the program:
   javac QuizApp.java
3. Run the program:
   java QuizApp

### Output
